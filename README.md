# SI Tracker App

「SI番号」単位で輸入商品の現在地や納期ステータスを管理するアプリです。

## 🧾 概要

輸入業務で扱う「スイッチインボイス番号（SI番号）」をキーに、以下の情報を一元管理します：

- 出荷日
- 到着予定日
- 船名
- 現在地（例：通関中、保税倉庫など）
- 遅延ステータス

## 💡 背景

現場での納期管理はメール・Excel・口頭のやり取りに依存しており、情報の抜け漏れ・確認工数が大きな課題です。  
本アプリはそれらを軽減し、輸入プロセスを視覚的に・正確に把握できることを目的としています。
